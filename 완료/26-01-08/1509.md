# 1509번: 팰린드롬 분할

## 문제 요약
### 문제
세준이는 어떤 문자열을 팰린드롬으로 분할하려고 한다. 예를 들어, ABACABA를 팰린드롬으로 분할하면, {A, B, A, C, A, B, A}, {A, BACAB, A}, {ABA, C, ABA}, {ABACABA}등이 있다.

분할의 개수의 최솟값을 출력하는 프로그램을 작성하시오.

### 사용 알고리즘
dp

### 입출력
- 입력: 첫째 줄에 문자열이 주어진다. 이 문자열은 알파벳 대문자로만 이루어져 있고, 최대 길이는 2,500이다.
- 출력: 첫째 줄에 팰린드롬 분할의 개수의 최솟값을 출력한다.
```
예제1

입력
BBCDDECAECBDABADDCEBACCCBDCAABDBADD
출력
22
```
## 풀이
dp를 두개 사용하는 문제이다. 첫번째는 펠린드롬을 구분하는 dp를 만들어야 한다. 모든 범위에서의 펠린드롬인지 확인한다. 다음은 분할 개수를 dp를 통해 알아보아야 한다. 주어진 길이까지 계속 dp를 통해 몇개로 분할할 수 있는지 구한다.

## 어려웠던 점
dp를 두개 쓰는데 하나는 1-based가 편하고 하나는 0-based가 편해서 둘이 섞여서 헷갈렸다. 각각 하는것도 어려웠던걸 보니 머리가 안돌아가는거 같다.

## 전체 코드
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

string s;

int dp[2501][2501];
int dpdp[2501];

int main() {
    ios::sync_with_stdio(false);
    cin.tie(NULL); cout.tie(NULL);

    cin >> s;

    for(int l = 1; l <= s.length(); l++){
        for (int i = 1; i <= s.length(); i++) {
            if(i+l-2 >= s.length()) break;
            if(l == 1){
                dp[i][i] = true;
            }
            else if(l == 2){
                if(s[i-1] == s[i]) dp[i][i+1] = true;
            }
            else{
                if(s[i-1] == s[i+l-2] && dp[i+1][i+l-2]) dp[i][i+l-1] = true;
            }
        }
    }

    dpdp[0]= 0;
    dpdp[1]= 1;
    for (int i = 2; i <= s.length(); i++) {
        dpdp[i] = 10000;
        for (int j = 0; j < i; j++) {
            if(dp[j+1][i]) dpdp[i] = min(dpdp[i], dpdp[j]+1);
        }
    }
    
    cout << dpdp[s.length()];


    return 0;
}
```
